# 快递驿站管理系统

小区门口快递驿站专用的极简管理系统。Python + Bottle + SQLite，单命令启动，开箱即用。

## 功能一览

- **📥 到件登记**：扫描单号，录入手机尾号、快递公司、货架号/层，自动生成6位取件码
- **📤 取件出库**：手机尾号 / 取件码 / 单号 三种方式查找，核验通过后标记出库；一个手机号多个件一起列出来
- **⏰ 超期提醒**：超过3天未取自动标红，可批量或一键全部发送取件提醒
- **🚚 代发快递**：录入收寄双方信息，自动按重量算运费，生成单号并打印面单
- **📊 统计查询**：今日入库/出库/积压数量一目了然；所有库存包裹按货架排序，支持关键字搜索和货架筛选，找件不用翻箱倒柜

## 技术栈

- 后端：Python 3.8+ / Bottle 0.12.x
- 数据库：SQLite（文件型，自动建库）
- 前端：原生 HTML + CSS + JavaScript（无框架）
- 测试：Python unittest（内置，无需额外安装）

## 快速开始

### 1. 安装依赖

```bash
cd d:\mf-06
pip install -r requirements.txt
```

只有 `bottle` 这一个依赖。

### 2. 启动服务

```bash
python app.py
```

服务起来后打开浏览器访问：**http://localhost:8080**

首次启动会自动在项目根目录创建 `express.db` 数据库文件。

### 3. 运行测试

```bash
python -m unittest tests.test_core -v
```

测试覆盖：
- 取件核验逻辑（手机尾号对/错、取件码对/错、已取走再取等场景）
- 超期判断逻辑（当天/1天/刚好3天/4天/30天）
- 运费计算（首重/续重/顺丰上浮）
- 完整入库→核验→出库工作流
- 单号重复入库校验

## 操作指南（从零走一遍）

### 一、录一个到件

1. 打开 http://localhost:8080
2. 点 **到件登记**
3. 依次填写：
   - 快递单号：比如 `SF20240621001`
   - 手机尾号：`1234`（必须4位数字）
   - 快递公司：选「顺丰」
   - 到件日期：默认今天就行
   - 货架号：`1`
   - 货架层：选「第2层」
4. 点 **📥 入库登记**
5. 页面提示「入库成功！取件码：XXXXXX，货架：1号2层」，OK了
6. 再多录几个，比如同一个尾号 `1234` 多录2个，后面取件时一起展示

### 二、取件走一遍

1. 点导航栏 **取件出库**
2. 默认是「📱 手机尾号」模式，输入 `1234`，点 **🔍 查找包裹**
3. 系统把尾号 `1234` 的所有待取件都列出来了，表格里能看到每个件的单号、取件码、**货架位置（几号架几层）**
4. 默认全选，在下方「核验方式」选「核验手机尾号」，输入 `1234`，点 **✅ 确认取件**
5. 跳转到「取件成功」页面，显示出库的包裹列表
6. 如果输入错的尾号（比如 `9999`），会提示「手机尾号不匹配」，不会出库
7. 也可以切换到「🔢 取件码」模式，用6位取件码取件，一样需要核验

### 三、模拟超期，看它标红提醒

超期阈值是 **3天**。需要手动模拟的话（二选一）：

**方法A：入库时把到件日期填早几天**
1. 到件登记页，把「到件日期」改成 5 天前的日期，其他正常填，入库
2. 点导航栏 **超期提醒**，这个包裹会被标红列出，显示「超期 X 天」
3. 也可以到 **统计/查件** 页面看，超期的行整个都是红底的
4. 勾选要催的件，点 **📨 向选中的发送提醒** 或 **📨 全部催取（一键群发）**，系统会写入 `reminders` 表（模拟短信发送）

**方法B：用 SQLite 工具改 `express.db`**
```sql
UPDATE packages SET arrival_date = date('now', '-10 days') WHERE id = 1;
```
改完刷新超期页面就能看到。

### 四、代发打单

1. 点 **代发快递**
2. 填寄件人、收件人信息（姓名、手机、地址各三项）
3. 选快递公司、填重量（比如 2.5 kg）
4. 点 **💴 计算运费**，系统会自动算出运费（首重1kg 10元 + 续重1.5kg × 5元 = 17.5元；顺丰再 ×1.3）
5. 信息确认无误，点 **✅ 确认下单并打单**
6. 跳转到面单预览页，点 **🖨️ 打印面单** 即可打印

## 运费规则

| 条件 | 价格 |
|------|------|
| 首重 ≤ 1kg | 10 元 |
| 续重每 kg | +5 元 |
| 顺丰 | 以上价格 × 1.3 |

例：2.3kg 中通 = 10 + 1.3×5 = 16.5 元；顺丰 = 16.5 × 1.3 = 21.45 元

## 超期规则

超过 **3天** 未取即为超期。可在 `modules/pickup.py` 顶部修改 `OVERDUE_DAYS` 常量调整。

## 端口

默认 `8080`，在 `app.py` 底部改。
